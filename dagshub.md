# DagsHub Repository - Data & Models

This folder contains:
- `outputs/class_distribution.csv - Class distribution metadata
- Dataset (if available) - CoralScapes 6-class dataset
- Models (if available) - Trained SegFormer models

## Data Overview

This DagsHub repo is used to store:
1. **Dataset**: CoralScapes remapped to 6 ecological classes
2. **Models**: Trained SegFormer-B5 model weights
3. **Metadata**: Class distributions and analysis files

## Dataset Classes (6 classes):
1. Water / Background
2. Live Coral
3. Bleached Coral
4. Dead Coral
5. Algae
6. Benthic Substrate
