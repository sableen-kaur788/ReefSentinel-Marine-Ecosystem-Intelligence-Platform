# ReefSentinel Backend

AI-powered coral reef analysis API using SegFormer.

## Features:
- Coral reef segmentation
- Reef health KPIs
- RAG-based AI chatbot
- Ecological report generation

## API Endpoints:
- `/predict` - Run segmentation on images/videos
- `/analyze` - Get full analysis
- `/rag` - RAG chatbot queries

Check out the main repo: https://github.com/sableen-kaur788/ReefSentinel-Marine-Ecosystem-Intelligence-Platform
